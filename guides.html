DevSecOps Fundamentals

Learn the basics of DevSecOps, including its principles, practices, and benefits.

  • Understand the importance of integrating security into the DevOps pipeline.
  • Learn about the DevSecOps lifecycle: Plan, Code, Build, Test, Release, Deploy, Operate, Monitor.
  • Explore key concepts like Shift-Left Security and Continuous Security.

SAST, SCA, IAC, Container Security

Learn about Static Application Security Testing (SAST), Software Composition Analysis (SCA), Infrastructure as Code (IAC) Security, and Container Security.

  • SAST: Analyze source code for vulnerabilities.
  • SCA: Identify vulnerabilities in third-party libraries.
  • IAC Security: Secure infrastructure code (e.g., Terraform, CloudFormation).
  • Container Security: Secure container images and runtime environments.

SAST with Fortify

Learn how to perform SAST scans using Fortify On Demand.

  • Set up Fortify On Demand for your projects.
  • Integrate Fortify with your CI/CD pipeline.
  • Analyze and remediate vulnerabilities found by Fortify.

IDE Security Plugins

Learn how to use IDE security plugins like SonarLint and Snyk.

  • SonarLint: Detect and fix code issues directly in your IDE.
  • Snyk: Find and fix vulnerabilities in dependencies.
  • Integrate these plugins into your development workflow.

Container Security with Snyk

Learn how to scan container images for vulnerabilities using Snyk.

  • Set up Snyk for container security.
  • Scan Docker images for vulnerabilities.
  • Integrate Snyk with your CI/CD pipeline.

DevSecOps with GitHub Actions

Learn how to implement an end-to-end DevSecOps pipeline using GitHub Actions.

  • Set up GitHub Actions for CI/CD.
  • Integrate security tools like SAST, SCA, and container security into the pipeline.
  • Automate security checks and deployments.

GitOps with Argo CD

Learn how to manage Kubernetes applications using GitOps with Argo CD.

  • Set up Argo CD for continuous deployment.
  • Sync Kubernetes manifests from a Git repository to your cluster.
  • Implement rollback and self-healing features.

Azure Kubernetes Service (AKS)

Learn how to deploy and manage containerized applications on AKS.

  • Set up AKS clusters and deploy applications.
  • Integrate AKS with Azure Security Center and Azure Policy.
  • Use Horizontal Pod Autoscaler (HPA) for automatic scaling.

Application Security

Learn the basics of application security and how it fits into DevSecOps.

  • Understand common application vulnerabilities (e.g., OWASP Top 10).
  • Learn how to secure APIs and web applications.
  • Explore tools and techniques for application security testing.

Advanced DevSecOps

Learn intermediate to advanced processes in DevSecOps.

  • Explore advanced CI/CD pipeline configurations.
  • Learn about threat modeling and risk assessment.
  • Understand how to implement security in microservices architectures.

DevSecOps CV

Learn how to create a CV for Security/DevSecOps jobs.

  • Highlight your DevSecOps skills and experience.
  • Include certifications, tools, and projects.
  • Tailor your CV for specific job roles.

DevSecOps Maturity Model

Learn about the DevSecOps Maturity Model and how to assess your organization's maturity level.

  • Understand the stages of DevSecOps maturity: Initial, Managed, Defined, Measured, Optimized.
  • Learn how to improve your organization's DevSecOps practices.

SCA with Snyk

Learn how to perform Software Composition Analysis (SCA) using Snyk.

  • Set up Snyk for dependency scanning.
  • Identify and fix vulnerabilities in third-party libraries.
  • Integrate Snyk with your CI/CD pipeline.

IAC Security with Checkov

Learn how to scan Infrastructure as Code (IAC) for security issues using Checkov.

  • Set up Checkov for IAC security.
  • Scan Terraform, CloudFormation, and other IAC templates.
  • Integrate Checkov with your CI/CD pipeline.

SAST with SonarQube

Learn how to run SAST scans using SonarQube with and without a CI/CD pipeline.

  • Set up SonarQube for code analysis.
  • Integrate SonarQube with your CI/CD pipeline.
  • Analyze and remediate vulnerabilities found by SonarQube.

DevSecOps Roles

Learn about the roles and responsibilities of a DevSecOps Engineer.

  • Understand the skills required for a DevSecOps role.
  • Learn about day-to-day tasks and responsibilities.
  • Explore career growth opportunities in DevSecOps.

SBOM and SCA

Learn about Software Bill of Materials (SBOM) and Software Composition Analysis (SCA).

  • Understand the importance of SBOM in supply chain security.
  • Learn how to generate and analyze SBOMs.
  • Explore tools for SCA and SBOM management.
